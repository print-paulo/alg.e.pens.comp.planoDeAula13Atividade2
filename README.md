# Algoritmos de Ordenação em C

Este projeto é uma aplicação de consola (CLI) desenvolvida em linguagem C que permite ao utilizador inserir números, armazená-los em ficheiros e ordená-los utilizando diferentes algoritmos clássicos de ordenação.

## Funcionalidades

O programa oferece as seguintes funcionalidades:

* **Entrada de Dados:** Permite inserir 6 números inteiros.
* **Persistência:** Guarda os números originais num ficheiro (`dados.txt`) e os ordenados noutro (`ordenado.txt`).
* **Visualização:** Lê e mostra o conteúdo do ficheiro de dados.
* **Edição:** Permite atualizar os valores armazenados no ficheiro original.
* **Algoritmos de Ordenação:**
    1.  Bubble Sort
    2.  Insertion Sort
    3.  Selection Sort
    4.  Merge Sort
    5.  Quick Sort

## Tecnologias Utilizadas

* Linguagem C
* Manipulação de Ficheiros (File Handling)
* Biblioteca Padrão (stdio.h, stdlib.h, string.h)
